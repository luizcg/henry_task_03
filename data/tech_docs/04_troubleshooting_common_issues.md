# Troubleshooting Common IT Issues - Acme Corporation IT

## Overview

This guide covers common IT issues employees encounter and the recommended first steps before contacting the IT Help Desk.

## Email Issues

### Cannot Send or Receive Email

1. Check internet connectivity by opening a web page.
2. Confirm you are signed in to Outlook with your Acme account.
3. If prompted, re-enter your password and complete MFA.
4. Check **Service Status** at status.office.com for known outages.
5. If the issue persists, restart Outlook or your computer.

### Mailbox Is Full

- Empty Deleted Items and Junk folders.
- Archive old emails to your Online Archive.
- Avoid sending large attachments; use OneDrive or SharePoint links instead.

## VPN and Network Issues

### Cannot Connect to VPN

- See the **VPN and Remote Access Guide** for detailed steps.
- Try a different network or restart your router.
- Verify that your password and MFA are working.

### Slow or Unstable Connection

- Close bandwidth-heavy apps (streaming, downloads).
- If on Wi-Fi, move closer to the router.
- For critical work, use a wired connection when possible.

## Laptop Performance Issues

### Slow Performance

- Restart your laptop (many users skip this).
- Close unused apps and browser tabs.
- Check for available disk space (keep at least 15â€“20% free).
- On Windows, run **Task Manager** to see high-CPU apps.

If performance issues persist, open a ticket with system details and recent changes (new apps installed, etc.).

### Application Crashes

- Update the application to the latest version.
- Reboot your laptop.
- Capture error messages or screenshots.

## Access and Permissions

### "Access Denied" to a File or Folder

- Confirm you are on VPN if accessing internal file shares.
- Verify you are using the correct path or link.
- Ask a colleague on the same team if they still have access.
- If you need access, request it via the appropriate access request form.

### Cannot Access a SaaS Application

- Check the **App Access Portal** for current status and access.
- Try logging out and back in via SSO.
- Clear browser cache or try an incognito/private window.

## Printing Issues (On-Site Offices)

### Cannot Print

- Confirm you are connected to the corporate network (Wi-Fi or wired).
- Ensure the correct printer is selected.
- Check the printer display for error messages (paper jam, low toner).

### Print Job Stuck in Queue

- Cancel the job from your device.
- Power cycle the printer if you have access.
- Try printing a test page.

## Collaboration Tools (Slack, Teams, Zoom)

### Audio or Video Problems

- Check if your microphone and camera are allowed in OS settings.
- Confirm the correct device is selected in the app.
- Use a headset to reduce echo and background noise.
- Quit and reopen the app if issues continue.

### Screen Sharing Not Working

- Ensure you have granted screen recording permissions (macOS).
- Try a different window or full-screen share.
- Restart the meeting app.

## Password and MFA Issues

See the **Account and Password Management Guide** for detailed steps.

### Common Problems

- Password rejected: may be expired or account locked.
- MFA code not received: check connectivity and authenticator app.
- Unexpected MFA prompt: deny and contact Security immediately.

## When to Contact IT

If you have tried the basic troubleshooting steps and still experience issues, open a ticket with IT. Include:

- A clear description of the problem
- When it started
- Steps to reproduce
- Screenshots or error messages
- Whether the issue affects only you or multiple people
